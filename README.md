# Self-Healing UI Automation via Robust UI-Element Similarity

## 📖 Project Overview

This project focuses on building a **context-aware visual change detection system** for UI testing.

Traditional UI automation approaches rely on DOM-based locators and pixel-wise comparison, which fail to capture contextual and structural UI relationships. This project reproduces a **graph-based, context-aware visual change detection framework** that:

- Detects UI controls using YOLOv5
- Constructs a spatial graph representation of UI layouts
- Applies recursive similarity-based graph matching
- Detects meaningful UI changes
- Visualizes detected changes via heatmaps
